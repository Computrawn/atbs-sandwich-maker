Sandwich Maker
Write a program that asks users for their sandwich preferences.
The program should use PyInputPlus to ensure that they enter valid input, such as:
Using inputMenu() for a bread type: wheat, white, or sourdough. (COMPLETE)
Using inputMenu() for a protein type: chicken, turkey, ham, or tofu. (COMPLETE)
Using inputYesNo() to ask if they want cheese. (COMPLETE)
If so, using inputMenu() to ask for a cheese type: cheddar, Swiss, or mozzarella. (COMPLETE)
Using inputYesNo() to ask if they want mayo, mustard, lettuce, or tomato. (COMPLETE)
Using inputInt() to ask how many sandwiches they want. Make sure this number is 1 or more. (COMPLETE)
Come up with prices for each of these options, and have your program display a total cost after
the user enters their selection. (COMPLETE)
